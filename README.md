# IEP 도구 모음

초등 특수학급 교사를 위한 개별화교육계획(IEP) 관련 도구 모음입니다.

| 도구 | 주소 | 설명 |
|---|---|---|
| IEP 초안 도우미 v3 | https://iepnote.github.io/iep/plan/ | 성취기준 기반 IEP 초안 작성 위저드 (주별 지도 계획·학교 양식 hwpx 자동 채움) |

## 사용 안내

- 페이지는 **암호화되어 있어 비밀번호가 필요**합니다. 비밀번호는 배포자에게 별도로 문의하세요.
- 모든 처리는 **브라우저 안에서만** 이루어집니다. 입력한 학생 정보는 어디에도 전송·저장되지 않으며,
  브라우저를 닫으면 사라집니다. 이어서 작업하려면 "작업 파일 저장(.json)"을 사용하세요.

## ⚠️ 꼭 지켜 주세요

1. **이 도구의 산출물은 "개별화교육지원팀 협의용 초안"입니다.** 최종 개별화교육계획은 반드시
   지원팀 협의와 교사의 전문적 판단을 거쳐 확정하고 나이스(NEIS)에 입력하세요.
2. **학생 실명 대신 가명·이니셜 사용을 권장**합니다.
3. **작업 파일(.json)에는 학생 정보가 들어 있습니다.** 메신저·클라우드 공유에 주의하고,
   이 저장소의 Issue·댓글에는 학생 정보(이름, 스크린샷 포함)를 절대 올리지 마세요.
4. hwpx 자동 채움은 특정 학교 양식 전용입니다. 다른 양식을 쓰는 학교는 "나이스 입력용 복사" 또는
   마크다운 저장 기능을 사용하세요. (양식팩 제작 문의는 배포자에게)

## 출처

- 성취기준: 2022 개정 특수교육 교육과정(기본) 및 공통 교육과정(교육부 고시)
- 주별 활동 예시: 2022 개정 특수교육 교과용도서의 단원·차시 구성을 참고하여 재구성한 초안 예시이며,
  교사의 수정·보완을 전제로 합니다.
